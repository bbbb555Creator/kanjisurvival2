# ⚡️ 漢字合体サバイバル

**「一瞬の迷いが命取り。50問連続正解を目指す、究極の漢字パズル。」**

表示されたパーツに、正しい部首を組み合わせて漢字を完成させるブラウザゲームです。
ミスった瞬間に最初からやり直しの「サバイバル形式」を採用。


---

### 🕹 遊び方
*   **① 準備**: `index.html` をブラウザで開くだけ！
*   **② 思考**: 画面中央の「パーツ」をじっと見る。
*   **③ 合体**: 4つの選択肢から、合体して漢字になる「部首」を選ぶ。
*   **④ 制覇**: 5段階の難易度を乗り越え、50問連続正解すればクリア！

> **※注意**：ミスると強烈な煽り（？）メッセージが出ますが、めげずに再挑戦してくださいw

---

### 🔥 難易度（全5ステージ）
1. **初級**: へん・つくりの基本問題
2. **中級**: かんむり・あしの上下パズル
3. **上級**: かまえ・たれ・しんにょう
4. **超級**: 難読・複雑な組み合わせ
5. **究極**: 漢字マスター専用の激ムズ常用漢字
