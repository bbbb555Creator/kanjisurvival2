漢字合体サバイバル
漢字のパーツを合体させて、正しい漢字を作るパズルゲームです。
ミスった瞬間に最初からやり直しの「サバイバル形式」！全50問、クリアできるかな？

🎮 遊び方
index.html をブラウザ（ChromeやSafariなど）で開く。

画面に出てくる「パーツ」を見て、合体できる「部首」を4つの中から選ぶ。

50問連続正解で完全クリア！

⚠️ 注意
1回でも間違えると、煽りメッセージと共にゲームオーバーになります（笑）。

音が出るので、音量設定には気をつけてね。
